# Roadmap

## Phase 1

- Replace template app with Open Scanner shell.
- Define Core Data entities for scans and pages.
- Build home list and scan detail views.
- Add plain PDF export and local persistence.

## Phase 2

- Integrate Vision OCR pipeline.
- Add scan text search and title suggestions.
- Export text in page order.
- Expand unit coverage around OCR mapping and preview logic.

## Phase 3

- Add VisionKit document scanning flow.
- Finish searchable PDF export polish.
- Improve simulator-friendly fallback handling.
- Add UI tests for core library flows.

## Phase 4

- Evaluate optional iCloud sync.
- Consider import from photos or PDFs.
- Explore annotations or lightweight editing.
